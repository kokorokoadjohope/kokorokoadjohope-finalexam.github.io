Education & Learning Experience

My educational journey has been a transformative experience that has shaped my understanding of technology and programming. From the moment I first encountered computers, I was fascinated by how software applications work and how websites are built. This curiosity led me to pursue courses in both C programming and web programming, where I have gained valuable skills and knowledge.

In my C programming course last semester, I learned the fundamentals of structured programming. I started with basic concepts such as variables, data types, and operators, then gradually moved to more complex topics like control structures, arrays, functions, pointers, and file handling. The course emphasized problem-solving skills and logical thinking, which are essential for any programmer. I practiced extensively on platforms like HackerRank and SoloLearn, completing numerous coding challenges and earning certificates that validate my skills.

This semester, I enrolled in the web programming course, which opened a whole new world of creativity and design. I learned HTML for structuring web content, CSS for styling and layout, and how to create responsive, user-friendly websites. The course covered everything from basic tags and attributes to advanced topics like flexbox, grid layouts, and media queries. I completed over ten exercises, each building on the previous one, and developed this portfolio website as my final project.

Beyond formal coursework, I have engaged in self-directed learning through online platforms, tutorials, and documentation. I believe that continuous learning is key in the fast-evolving field of technology. My goal is to become a proficient full-stack developer, capable of building robust applications that solve real-world problems. This portfolio represents not just my academic work, but my dedication to mastering the craft of programming.

Through both my C programming and web programming courses, I have developed a strong foundation in computational thinking, attention to detail, and the ability to debug and optimize code. These skills will serve me well as I continue my journey in computer science and software development.
